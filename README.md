## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Generate Database

npx prisma generate

npx @better-auth/cli generate

npx prisma db push

(Opt : npx prisma studio)
